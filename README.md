# Al Ain Properties — Premium Real Estate Platform

A modern, bilingual (Arabic/English) real estate website for Al Ain, UAE.

## 🌟 Features

- **Bilingual** — Full Arabic & English support with RTL layout
- **Dark Mode** — Elegant white/blue/gold palette in both modes
- **Property Listings** — Villas, apartments, shops, offices, warehouses, farms, land
- **Advanced Search** — Filter by location, price, type, bedrooms, bathrooms, size, furnished status
- **9 Al Ain Areas** — Al Jimi, Al Towayya, Al Mutaredh, Al Hili, Al Maqam, Zakher, Al Foah, Falaj Hazza, Al Yahar
- **Property Details** — Photo gallery, description, features, OpenStreetMap location
- **WhatsApp Integration** — Floating button, inquiry WhatsApp deep-links, viewing booking → WhatsApp
- **Admin Dashboard** — Full CRUD for properties, news, inquiries, viewing bookings
- **News Section** — Real estate news with categories
- **SEO-Ready** — Metadata, semantic HTML, fast loading
- **Mobile-First** — Fully responsive

## 🛠️ Tech Stack

- Next.js 16 (App Router)
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui (Radix UI)
- Prisma ORM (SQLite)
- next-themes (dark mode)
- Custom i18n (no external dependency)

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Push database schema
npm run db:push

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

The first visit auto-seeds:
- 21 sample properties across all 9 Al Ain areas
- 8 news articles about UAE real estate
- Default admin account
